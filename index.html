Taxi Ride Sharing

Producing the shortest and cost effective route

Download .zip Download .tar.gz View on GitHub

Taxi Ride Sharing is an academic project

This is a novel algorithm, which will not only provide a shortest path and the shared path for the ride but also gives the cost effective path. Here two users from different sources with same destination will be given a shared path which is shortest as well as cost effective.
